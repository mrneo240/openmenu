Github Release Message (openMenu):
# Public alpha Release (2021-12-5)

**New Features:**
* using GDMENU loader for higher compatibility of gd-rom + cdi
* custom user themes + saving theme settings
* manually fix bad serial info from sega

## General Controls:
A - Launch
START - Open preferences menu
Y - return to bios

## Specific UI Controls:
**LineDesc:**
LEFT - 1 item left
RIGHT - 1 item right
UP - 10 items left
DOWN - 10 items right
LTRIG - 25 items left
RTRIG - 25 items right

**Grid3:**
DPAD - Navigation
LTRIG - Page up
RTRIG - Page down
X - Show large cover art

**GDMENU:**
UP - Previous Item
DOWN - Next Item
LTRIG - Move 5 Items Up
RTRIG - Move 5 Items Down

## Installation (Read before attempting):
* Install [GDMENUCardManager](https://github.com/mrneo240/GDMENUCardManager) as per [Instructions](https://github.com/mrneo240/GDMENUCardManager/wiki/Instructions)
* Delete all files under in `tools/menu_data`  folder where you installed [GDMENUCardManager](https://github.com/mrneo240/GDMENUCardManager)
* Extract all files from this release `tools/menu_data` and ensure you also extract the **[Asset Pack](https://www.mediafire.com/folder/ocj3noyzuq5t7/DAT_latest)** into the same folder
  *  Ensure you have `1ST_READ.BIN`, `META.DAT`, `BOX.DAT`, `ICON.DAT`, `EMPTY.PVR` files and the `font` and `theme` folders.
* Use GDMENUCardManager normally

## Notes:
* Current Metadata DB and Boxart DB should feature completed NTSC-U, PAL & Atomiswave games.
* If you have an NTSC or PAL game that is missing Boxart or Metadata please file an issue here!
* Work has started to add JAP & Homebrew games Boxart + Metadata.

## Issues:
* Some users are reporting an issue with black screen and freeze. If this happens please turn off console, remove VMU from controller 1 and try again. 
  * Please also file an issue describing your setup and what happened. This is a very tricky bug affecting a very small percent of users.
